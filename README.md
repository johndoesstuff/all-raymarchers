# All Raymarchers

Programming a raymarcher is a good way to learn a language, it involves a decent amount of complexity while not being overly ambitious. I would like to dip my toes into more programming languages and so as a fun side challenge I'd like to grow a collection of raymarchers that have exactly the same final output:

```
################################  ..::::::::..    
################################  ...:::::...     
###############################  .........     |||
##############################| ........  ||||||||
#############################.    ...   ||||||||||
###########################|:         :|||||||||||
:########################|||.      ::::|||||||||||
::::::###############|||||||     .::::::||||||||||
:::::::::.....||||||||||||:     ...::::::|||||||||
 ::::::......     .::::::.       ....::::::       
                                                  
                                                  
       ......       ......       ......           
::::::::......   :::::::::.     ...:::::::::   ...
:::::::::....  ||||||||||::      .::::::::||||||||
:::::::::   :||||||||||||||.       :::::||||||||||
:::::::.   |||||||||||||||||        :::|||||||||||
:.        :|||||||||||||||||           |||||||||||
    .##################|||||     ...... ||||||||||
 |########################|      ......  .||||||||
```

Use `compare.py` to compare against `goal.txt`, note that some of these are off by a character or 2. If you feel like you can improve these feel free to try!
